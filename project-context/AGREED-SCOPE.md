## The product in one line

A complete redesign of **cloudtitle.com** on **WordPress + Divi**, positioning Cloud Title as Florida's #1 AI-powered title agency with **Safe Listing™** as the flagship acquisition funnel.

## Engagement shape

- **One-time build** — no retainer, no ongoing ops
- **Target timeline:** 8 weeks from kickoff (4–5 weeks possible if Client returns approvals in 24–48h)
- **Platform locked:** WordPress + Divi (existing). Do not propose alternatives.
- Work begins only after kickoff payment is confirmed by Mohammad

---

## Pages we are building (full IA)

1. **Home** — copy already supplied verbatim (see `briefs-from-thomas/Complete Homepage Copy – Divi Implementation Version.docx`)
2. **Safe Listing™** — flagship product page (Client supplying most copy + video, 1–2 weeks out)
3. **Agents** (parent page + 4 subpages)
   - Start Safe Listing
   - Submit Title Order
   - Rush Commitment
   - Marketing Resources
4. **Buyers & Sellers**
5. **Resources** (parent page + subpages)
   - Blog (Cloud Title Intelligence™)
   - FAQ
   - Case Studies
6. **About**
7. **Contact**
8. **Legal pages** — preserved verbatim, do not alter

Sticky navigation with persistent **"Start Safe Listing"** CTA right-aligned.

## Design process

- **3 homepage concepts in Figma** delivered before any Divi build
- Client picks a direction; first revision applied
- Page-by-page build with **2 revision rounds per page** (hard cap — don't exceed)
- Mobile-first, tested at 375 / 768 / 1024 / 1440 breakpoints

## Blog configuration (built as part of this engagement)

- Grid layout
- 5 categories
- RankMath SEO plugin configured
- CTA blocks on posts
- Author attribution
- Social sharing
- Client provides the seed posts

## SEO + AI (one-time setup, folded into the build)

- Schema.org markup across all pages
- RankMath full configuration
- `llms.txt` for AI crawlers
- Core Web Vitals tuning — target sub-2.5s load
- GA4 custom events
- Sitemap submission after launch

## CTAs and forms

**Every CTA on every page must connect to a GoHighLevel form.** Client to provide GHL form embeds/IDs by Week 3 of the build. Don't ship a CTA that goes anywhere else.

## Out of scope (do not build)

- Auto-blog engine / managed content posting
- Inside-sales follow-up CRM system (Client building this themselves)
- Ongoing SEO retainer or content production
- Platform migration off WordPress + Divi
- Anything not in the page list above
